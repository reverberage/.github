# reverberage/.github

Shared community health files and templates for the reverberage organization.

## Contents

- **CONTRIBUTING.md**: Contribution guidelines for all repos
- **SECURITY.md**: Security policy for all repos
- **CODE_OF_CONDUCT.md**: Code of conduct for all repos
- **ISSUE_TEMPLATE/**: Issue templates for bug reports and feature requests
- **PULL_REQUEST_TEMPLATE.md**: Pull request template

## Usage

These files are automatically applied to all repositories in the reverberage organization.

## Alignment with Alibaba Plan

Centralized community health files ensure consistency across all repos and demonstrate project maturity for Stage B/C applications.
