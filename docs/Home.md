# Welcome to reverberage Shared Config

This `.github` repository holds shared GitHub configurations, reusable Actions, and organization-level automation for the reverberage ecosystem.

## What's Here

- **`.github/actions/`** — reusable composite GitHub Actions used across all repos
- **`.github/ISSUE_TEMPLATE/`** — organization-wide issue and PR templates
- **Shared workflows** — patterns and config applicable to all satellites

## Key Actions

| Action | Description |
|--------|-------------|
| `sync-wiki` | Syncs `docs/` markdown to repository wiki on push |
