# .github

Profile, community health files, and organization-level templates for the reverberage organization.

## What this repo contains

- **Profile README**: `profile/README.md` — shown on the organization page
- **Issue templates**: `.github/ISSUE_TEMPLATE/` — standardized issue forms
- **PR templates**: `.github/PULL_REQUEST_TEMPLATE/` — standardized PR descriptions
- **Community health**: `CONTRIBUTING.md`, `SECURITY.md`, `CODE_OF_CONDUCT.md` — shared across all repos
- **Organization workflows**: `.github/workflows/` — shared GitHub Actions

## Organization Config

- **Dependabot**: Automated dependency updates across all satellites
- **Stale bot**: Marks inactive issues/PRs
- **Label sync**: Consistent labeling across repos
